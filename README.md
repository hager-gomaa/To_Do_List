# To-Do List Application (Python)

This is a simple command-line To-Do List application developed using Python.
The project was created during my first year at university as a practice
to understand basic programming concepts.

## Project Description
The application allows users to manage a list of tasks through a text-based menu.
Users can add tasks, view all tasks, remove tasks, or exit the program.

## Features
- Add new tasks to the list
- View all existing tasks
- Remove tasks by their number
- User-friendly command-line menu
- Input validation for better user experience

## Technologies Used
- Python
- Jupyter Notebook / Python Script
